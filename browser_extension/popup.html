HLA FASTA Reformatter

This plugin is designed to be used with the EMBL-EBI Sequence Alignment Tool

To use this tool:
  1. Go to the EMBL-EBI Sequence Alignment Tool
  2. Enter your data for Steps 1&2
  3. Select Plain text as the output (quicker), however both formats work
  4. Select Align Sequences Now and wait for the results page to fully load
  5. Click the Download FASTA Data button that appears at the top of the page
